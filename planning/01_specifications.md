## main features
* Homework help agents for all subjects
* Multi-turn tutoring with saved session state
* Schedule coordination for school work and activities
* User interface of chat and voice
* Watches a folder for new homework files (PDF, DOCX, TXT, images)
* Checks your email inbox for new homework emails (Gmail/IMAP)
* Automatically extracts text and creates a study plan
* Spawn subject-specific tutor agents that share a common knowledge base
* Implements a web UI on localhost
* Runs on 12 GB GPU via Ollama
* Zero cloud, zero cost, zero internet after setup


## connectivity
Data should feed in to the agents context through multiple inputs
* emails sent to the agent
* google drive shared with the agent
* google calendar shared with the agent

## logging
* any email, google drive or other access should be logged
* API access should be logged

## reporting
* daily reports of updates to calendar and google drive
* daily reports of emails and additional files ingested
* system will generate reports of usage of paid resources
